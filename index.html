CitrusCSS

The GitHub

A lightweight, citrus-themed stylesheet for clean, fresh, and modern HTML styling.

Use it for simple static websites or as a foundation for custom styles. It is not a full CSS framework.

Features

  • Minimalist and lightweight
  • Clean, citrus-inspired design
  • Modern Dark and light theme
  • Accessible and semantic styling

Why CitrusCSS?

Created to bring a fresh, citrusy vibe to web design, perfect for developers who love simplicity and style.

  1. Streamlines website styling
  2. Embraces citrus aesthetics
  3. Easy to customize

How to Use

Add the stylesheet:

Download Citrus.css from GitHub into your repo and link it up in the head of your HTML.

<link rel="stylesheet" href="citrus.css">
